基本認識

- 資料型別
int - 整數 ex:1,2,3
str - 字串 ex:"我是個字串"
float - 浮點數 ex:0.01
bool - 布林值 ex:True
list - 列表 ex:[1,2,3]
tuple - 元組 ex:(1,2,3)
dict - 字典 ex:{'apple':3,'banana':4}
set - 集合 ex{1,2,3}
以上為常見的資料型別 細項之後會介紹

- 運算字符
() - 優先計算
** - 次方
* - 乘法
/ - 除法
// - 整除
% - 求餘數
+ - 加法
- - 減法
> - 大於
< - 小於
== - 相等
<= - 小於等於
>= - 大於等於
!= - 不等於
and - 雙方都需等於True才會輸出True
or - 只要有一方等於True就會輸出True
not - 相反
以上按照優先級排序
